Marble Solitaire

This marble solitaire game was made in Java for Northeastern's CS 3500 course. It features three possible game modes you can choose from, and when a player starts the game they are presented with an initial board in which they can enter to and from coordinates to make moves and play the game. The game is over when there are no more possible moves left, and the objective of the game is to clear as many marbles as possible.
